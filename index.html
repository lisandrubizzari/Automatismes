Mode d'emploi

Choisis un thème, lance une série de 10 questions et suis la barre de progression pour rester concentré. Le mode clair/sombre, le mode test de 5 minutes et les résumés motivants t'accompagnent partout.

  • Questions inédites à chaque session avec correction immédiate.
  • Explications claires, accessibles clavier et compatibles mobile.
  • Résumé final avec score, temps écoulé et conseils.

Conseils de réussite

Bloque 5 minutes par thème, active le mode test une fois par jour et note tes scores pour visualiser ta progression.

  • Lis les explications même lorsque tu réussis la question.
  • Utilise le mode sombre la nuit pour limiter la fatigue visuelle.
  • Partage tes retours sur GitHub pour faire évoluer l'outil.
